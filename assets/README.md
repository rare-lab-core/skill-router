# assets

Put the README banner image here, named `banner.png` (or `banner.webp`, `banner.jpg`).

Then add this line back at the top of the main `README.md`, inside the opening
`<div align="center">`, where the placeholder comment sits:

```html
<img src="assets/banner.png" alt="skill-router" width="680">
```

Recommended width 680px to match the header layout.
